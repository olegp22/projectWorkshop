# 🎯 Сервис сбора экспертной обратной связи по проектам

## 📖 Описание проекта

Сервис предназначен для организации проверки и оценки проектов студентов, участников конкурсов и других пользователей. Платформа позволяет загружать работы, автоматически распределять их между проверяющими, собирать текстовую и голосовую обратную связь, а также формировать результаты в структурированном виде.

Проект решает проблему хаотичного сбора обратной связи, потери комментариев и отсутствия прозрачности процесса проверки.

## ✨ Основные возможности

- 🔗 Создание групп и приглашение участников по инвайт-ссылкам
- 🎮 Поддержка трёх режимов проверки
- 📤 Загрузка файлов и ссылок на работы
- 🤖 Автоматическое распределение работ между проверяющими
- 💬 Текстовые и голосовые комментарии
- 🎙️ Расшифровка голосовых комментариев в текст
- 📊 Формирование сводных таблиц и рейтингов
- 📎 Экспорт результатов в CSV / Excel
- 📱 Адаптивный интерфейс для компьютера и телефона

## 🎮 Режимы работы

### 1. Экспертная проверка

Преподаватель или внешний эксперт проверяет студенческие работы.

**Процесс работы**
- Организатор создаёт группу
- Приглашает студентов и экспертов
- Настраивает критерии оценки
- Студенты загружают работы
- Эксперты получают работы из очереди
- Эксперты оставляют оценки и комментарии
- Студенты и организатор получают результаты проверки

**Особенности**
- ✅ Проверка по критериям
- 🎙️ Голосовые и текстовые комментарии
- 📋 Очередь непроверенных работ
- 🚫 Исключение одновременной проверки одной работы несколькими экспертами

### 2. Peer-to-peer оценка

Студенты проверяют работы друг друга.

**Процесс работы**
- Организатор создаёт группу
- Настраивает критерии оценки
- Студенты загружают работы
- Система автоматически распределяет N работ каждому студенту
- Студенты оценивают работы по шкале от 0 до 10
- Система агрегирует результаты проверки

**Особенности**
- 🔄 Автораспределение работ
- 🛡️ Защита от самопроверки
- 🔁 Отсутствие дублирования проверок
- 📈 Подсчёт среднего балла и разброса оценок

### 3. Конкурсное голосование

Жюри или участники оценивают проекты.

**Процесс работы**
- Организатор создаёт конкурсную группу
- Настраивает критерии и шкалу оценки
- Приглашает участников и жюри
- Участники загружают проекты
- Жюри оценивает работы
- Система формирует итоговый рейтинг

**Особенности**
- ⚙️ Гибкая настройка шкалы оценки
- 🏆 Формирование рейтинга проектов
- 💬 Комментарии к проектам
- 📋 Сводные результаты конкурса

## 🧠 Бизнес-логика

### Автораспределение
- 🚫 Работа не может быть назначена своему автору
- 🔁 Исключается дублирование проверок
- 🔢 Ограничивается количество проверок на одного пользователя

### Очередь проверок
- 👁️ Проверяющий видит только доступные ему работы
- ✅ Проверенные работы исключаются из очереди
- 📊 Поддерживаются статусы проверки
